# goto

> `AGENTS.md` is the project guidance file for this repository.

Two install surfaces for macOS project jumping:
**`goto`** (CLI picker) and **`Goto.app`** (menu bar UI).

## Tech stack

- **CLI** (`goto`): Node.js >= 20, zero dependencies, ESM (`"type": "module"`)
- **Native App** (`Goto.app`): Swift 5.8, SwiftUI, macOS 13+ â€” Xcode project (`product/macos/Goto.xcodeproj`), generated by Ruby script

## Architecture rules

1. `goto` stays separate from `Goto.app`.
2. `Goto.app` is an agent-style app (`LSUIElement = true`) that lives in the menu bar and opens a settings window on demand.
3. `Goto.app` watches `~/.goto` and keeps the menu bar project list in sync automatically.
4. Terminal auto-detection uses `NSWorkspace.runningApplications` and is configurable via `UserDefaults`.
5. The Xcode project is generated by `scripts/generate_macos_project.rb`; new native source files must be registered there.
