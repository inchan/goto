# Cleanup 2026-05-06

The cleanup pass removed drift between the CLI and the app menu bar by deleting the CLI-local copies of sorting, recent-project, display-name, and path-shortening logic. `GotoCLI/main.swift` now delegates those behaviors to `GotoProjectList` and `GotoSettings` from the shared module.

The Settings window no longer stores popup and checkbox references that were only assigned and never read. The retained state is limited to the window and the terminal status label, which are actually needed after setup.

Root-level generated app bundles were removed from the repository workspace: `Goto.app` and `GotoLauncher.app`. The stale root `ROADMAP.md` was also removed because its validation counts no longer matched the current XCTest suite. Durable project documentation now lives under `wiki/`.

README was updated with the current CLI recent behavior, sort config files, menu bar grouping option, and tooltip-only path display.

Validation performed after cleanup:

- `xcodebuild -project Goto.xcodeproj -scheme Goto -configuration Debug build`
- `xcodebuild -project Goto.xcodeproj -scheme GotoCLI -configuration Debug build`
